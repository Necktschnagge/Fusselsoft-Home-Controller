59 how to use this library:
60 
61 This library offers two main things.
62 I creating / handling a SysTime object
63 II activate / deactivate 16bit Timer1 of the controller and link it to a SysTime object.
64 
65 (I) is represented by all non-static members of class SysTime, (II) by all static ones.
66 
67 Steps to do, in order to build a running SysTime clock:
68 1 First create a SysTime object
69 (! check ctor description and check error information returned )
70 
71 set min_compare_match_value if necessary. // ## was ist das???
72 
73 2 use the (static) link(..) method to tell the timer engine about your SysTime object.
74 it gets a persistent reference to your object.
75 3 start() the timer with static start() method
76 (! check the return value of start() and it's description of course )
77 4 Get the sysTime value whenever you want by using operator() of your SysTime object.
78 e.g.
79 time::EMT before = my_sysTime();
80 some_method_which_needs_some_time();
81 auto after my_sysTime();
82 // duration of function: after - before
83 
84 5
85 There are a few other methods you may use, but in normal / easy cases,
86 where you just want to start a SysTime once and recurrently get the now time,
